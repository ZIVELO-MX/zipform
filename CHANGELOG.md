# Changelog

All notable changes to Zipform are documented in this file.

## [1.0.0] - 2026-07-04

### Added

- Unified dashboard shell, shared UI package, authentication, and responsive navigation.
- TLOZ dashboard, board, list, table, calendar, project, inventory, and mission detail workflows.
- PostgreSQL persistence with Prisma, Supabase pooled/direct connections, migrations, and integration tests.
- Vercel production and Preview deployments through the native Git integration.

### Changed

- Established a clean TLOZ baseline with Koda, Fidelity, TLOZ, and Web Corporativa projects.
- Moved the Quotes integration target to version 1.1.0.
